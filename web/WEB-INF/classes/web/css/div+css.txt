div+css

1.css	层叠样式表单
	div用于存放内容的容器
	css指定div中的内容如何显示，包括位置和外观

	内部css	写在html文件内部
	外部css	单独写一个文件

2.滤镜
3.css选择器
	类选择器	可以有多个类选择器	class="style1 style2" 在css文件中最后出现的为准
	id选择器	最多只有一个id选择器
	html选择器
	通配符选择器
	赋值选择器
		优先级：id > 类 > html

4.块元素和行内元素（内联元素，inline）
	块元素 div、p
		不管内容多少，都要换行并且占满整行；
		块元素可以放文本和块元素
	行内元素 span、a、input等
		只占内容宽度，默认不会换行；
		行内元素一般放文本或者其他行内元素；
	一些css对行内元素不生效，margin、left、right
	块元素可以以行内元素的形式显示，行内元素可以以块元素进行显示

5.流
	流：元素（标签）的排列顺序，分为标准流和非标准流

7.浮动
	左浮动:尽量向左移动，让出右边空间给下一个元素显示
	右浮动:元素尽量向右移动，直到碰到父元素的右边界

	默认情况下div纵向排列
